rez-pip
=======

rez-pip is a rez_ command-line plugin that you can use to make package from the `Python Package Index`_ and other indexes available to your rez package ecosystem.

.. _rez: https://github.com/AcademySoftwareFoundation/rez
.. _Python Package Index: https://pypi.org
